== Description ==

3dpieces is a custom WordPress theme designed specifically for 3D printing and CAD design services.

Features:
* Dark theme with electric blue accents
* Fully responsive design
* Custom portfolio post type
* Portfolio filtering
* Image lightbox
* Contact form integration
* WhatsApp integration
* Smooth scroll animations
* Mobile-optimized navigation
* SEO-friendly structure

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through the 'Appearance' menu in WordPress
3. Create pages using the included templates
4. Configure menus and customize settings

== Frequently Asked Questions ==

= How do I change the accent color? =

Edit the --color-accent variable in assets/css/variables.css

= How do I add portfolio projects? =

Go to WordPress Admin → Portfolio → Add New. Upload images, set categories, and publish.

= How do I customize contact information? =

Update footer.php and page-contatti.php with your email and WhatsApp number.
